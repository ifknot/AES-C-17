Detailed Description

template<ROUNDS R = R256, KEY_LENGTH N = N256, typename T = uint8_t>
class crypto::aes::encrypt< R, N, T >

AES block cipher encrypt functor implementation available choices are AES128, AES192, AES256. The implementation is verified against the test vectors in: National Institute of Standards and Technology Special Publication 800-38A 2001 ED.

Template Parameters
Rnumber of round keys needed (AES-256 default 15)
Nlength of the key in 32-bit words (AES-256 default 8)
T8-bit type (default uint8_t) manipulating 8-bit bytes obviates the need to handle endianness across platforms.
