◆ nonce_mode_t

enum crypto::nonce_mode_t

Nonce modes, two basic groups

  • Cryptographically secure (for a stream cipher, that is resistant to nonce reuse) using block_t = aes::encrypt<>::block_t; using value_type = aes::encrypt<>::value_type;
  • Pseudo-random (vulnerable to nonce reuse)
    Note
    Given a secure random number generator, one can _(almost)_ guarantee to never repeat a nonce twice in a lifetime.
